# 🦺 İSG CEP - İş Sağlığı ve Güvenliği Eğitim Platformu

İş Sağlığı ve Güvenliği (İSG) alanında kapsamlı bir eğitim platformu. Konu anlatımları, interaktif testler ve sınav hazırlığı ile İSG bilgini test et, geliştir ve sertifikaya hazırlan!

![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue)
![License](https://img.shields.io/badge/License-MIT-green)
![Mobile](https://img.shields.io/badge/Mobile-Responsive-orange)

## 🌐 Canlı Demo

👉 [https://isgcep-ux.github.io/isgcepapp/](https://isgcep-ux.github.io/isgcepapp/)

## ✨ Özellikler

- 📚 **10 Kapsamlı Konu Anlatımı**
  - İSG Mevzuatı
  - Risk Değerlendirmesi
  - Kişisel Koruyucu Donanımlar
  - Yangın Güvenliği
  - Elektrik Güvenliği
  - Kimyasal Maddeler
  - İlkyardım
  - Ergonomi
  - İş Kazaları
  - Meslek Hastalıkları

- 📝 **20+ Soruluk İnteraktif Test**
  - Anlık doğru/yanlış geri bildirimi
  - Otomatik puan hesaplama
  - Başarı yüzdesi
  - Tekrar deneme özelliği

- 🎨 **Modern Tasarım**
  - Mobil uyumlu (responsive)
  - Karanlık/Aydınlık mod
  - Smooth animasyonlar
  - Kullanıcı dostu arayüz

- 🚀 **Teknik Özellikler**
  - Tek HTML dosyası (kurulum gerektirmez)
  - Harici bağımlılık yok
  - Hızlı yükleme
  - GitHub Pages ile ücretsiz yayın

## 🛠️ Teknolojiler

- **HTML5** - Yapı
- **CSS3** - Stil ve animasyonlar
- **Vanilla JavaScript** - İnteraktif özellikler
- **GitHub Pages** - Hosting

## 📦 Kurulum

### Yöntem 1: Doğrudan Kullanım
1. `index.html` dosyasını indirin
2. Tarayıcınızda açın
3. Kullanmaya başlayın!

### Yöntem 2: GitHub Pages ile Yayınla
```bash
# Repoyu klonlayın
git clone https://github.com/isgcep-ux/isgcepapp.git

# Dizine gidin
cd isgcepapp

# GitHub Pages'i aktif edin
# Settings → Pages → Branch: main → Save
```

## 📖 Kullanım

### Konu Anlatımları
1. Ana sayfada "Konulara Başla" butonuna tıklayın
2. İstediğiniz konuyu seçin
3. Detaylı açıklamayı okuyun
4. Modal pencereyi kapatmak için ✕ butonuna tıklayın

### Sınav Modu
1. "Sınava Gir" butonuna tıklayın
2. Soruları cevaplayın
3. Her sorudan sonra doğru/yanlış geri bildirimi alın
4. Sonuç ekranında başarı puanınızı görün
5. "Tekrar Dene" ile sınavı baştan başlatabilirsiniz

### Tema Değiştirme
- Sağ üstteki "🌙 Tema" butonuna tıklayarak karanlık/aydınlık mod arasında geçiş yapın

## 📊 İçerik Detayları

### Konular
| # | Konu | Açıklama |
|---|------|----------|
| 1 | İSG Mevzuatı | 6331 sayılı kanun, temel ilkeler |
| 2 | Risk Değerlendirmesi | Tehlike tanımlama, analiz, kontrol |
| 3 | KKD | Baret, eldiven, gözlük, ayakkabı |
| 4 | Yangın Güvenliği | Yangın sınıfları, söndürme, tahliye |
| 5 | Elektrik Güvenliği | Çarpılma, topraklama, koruma |
| 6 | Kimyasal Maddeler | MSDS, etiketleme, depolama |
| 7 | İlkyardım | CPR, kanama, yanık, zehirlenme |
| 8 | Ergonomi | Oturma, kaldırma, çalışma ortamı |
| 9 | İş Kazaları | Nedenler, raporlama, önleme |
| 10 | Meslek Hastalıkları | Pnömokonyoz, asbestozis, korunma |

### Soru Bankası
- 20 çoktan seçmeli soru
- 4 şık (A, B, C, D)
- Konu bazlı dağılım
- Zorluk seviyeleri karışık

## 🎯 Hedef Kitle

- İSG sertifikası almak isteyenler
